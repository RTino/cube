Why?

In a fast growing company like Zalando, where the size of the technology team tripled in 2012, it is becoming increasingly difficult to remember peopleâ€™s names, right? Weeks will pass before a newcomer knows who is doing what and where. Initially, we developed the Cube to solve this problem, but we later realized that the actual problem was more abstract. Having a bunch of CSV, JSON, and XML files and searching important information in them motivated us to fill the Cube not only with our technology team but also with all of our projects, hosts, and many other things.

Installation

Just follow the next simple steps and you'll get the example up and running

  1. Get it from our GitHub repo

    $ git clone git@github.com:zalando/cube.git

  2. Install dependencies

    $ cd cube
    $ npm install .

  3. Configure

    $ vi server.settings.coffee

  4. Run Solr instance on a new terminal

    $ cd cube/solr
    $ mvn clean install tomcat:run-war -Pdevelopment

  5. Start your app

    $ coffee cube.coffe
