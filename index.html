What is it?

The Cube is a versatile crud-like management tool that allows you to organize and manage your data with ease. As long as you can write as Schema for it, you'll be able to get important information from your data by using faceted queries, string filtered and sorted results, and multiple other features. It started as a tool to integrate a fast-growing department from Zalando, but it now has become a data hungry indexer monster.

Installation

Just follow the next simple steps and you'll get the example up and running

  1. Get it from our GitHub repo

    $ git clone git@github.com:zalando/cube.git

  2. Install dependencies

    $ cd cube
    $ npm install .

  3. Configure

    $ vi server.settings.coffee

  4. Run Solr instance on a new terminal

    $ cd cube/solr
    $ mvn clean install tomcat:run-war -Pdevelopment

  5. Start your app

    $ coffee cube.coffe
