Importing external data to a Cube.

The Cube has a built-in importer framework which you can use to create your own data importers. It comes with built-in importers for JSON, CSV and XML external data.

The built-in ImportManager is responsible for reading import.json files and scheduling these importers to run at the specified intervals.

The import.json file

Importer jobs are defined inside a import.json file, located under the desired Cube entity folder.

Creating your own custom importer

If you wish to make a custom importer, for instance to get and transform data from a SQL database, you'll need to create a xxxImporter.coffee file under the /server/importers/ directory.

Monitoring importer jobs

The easiest way to view the status of importer jobs is by opening /importers on your browser.
