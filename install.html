Three steps to get the Cube up and running!

  1. Resolve dependencies

    An installation script is provided to help you solve the dependencies and download the latest version of the Cube app. Only the following operating systems are supported: Linux Ubuntu, Debian, Fedora, CentOS, Red Hat, and Mac OS X. If yours is not on the list, proceed to install manually. Please note that JDK (>= 1.6) is a requirement which won't be solved by this script.

    Script
    $ bash <(curl -s http://zalando.github.io/cube/install.sh) && cd cube
    Manual

    Requirements: Maven (2.2), NodeJs (>=0.8) and ImageMagick (>=6.8.0)

    Download the cube app zipfile from https://github.com/zalando/cube/archive/master.zip 

    $ unzip cube-master.zip
    $ mv cube-master cube
    $ cd cube
    $ npm install .
  2. Set up the Solr Database

    Build and run the Solr Database

    $ cd solr
    $ mvn clean install tomcat:run-war -Pdevelopment
  3. Start the Cube App

    Start the Cube nodejs server on a new terminal, from the cube project directory

    $ coffee cube.coffee

    Open it in your favorite browser

    http://localhost:3000
