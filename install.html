Installation

The Four steps to get the Cube up and running!

  1. Resolve dependencies

    There are two ways to get the project installed: the easy way and the fun way. For both you need to have a working java environment with JDK (>= 1.6).

    Easy - installation script
    $ bash <(curl -s http://zalando.github.io/cube/install.sh) && cd cube
    Fun - Manual install

    Requirements: Maven (2.2), NodeJs (>=0.8) and ImageMagick (>=6.8.0)

    Download the cube app zipfile from https://github.com/zalando/cube/archive/master.zip 

    $ unzip cube-master.zip
    $ mv cube-master cube
    $ cd cube
    $ npm install .
  2. Set up the Solr Database

    Build and run the Solr Database

    $ cd solr
    $ mvn clean install tomcat:run-war -Pdevelopment

  3. Start the Cube nodejs server on a new terminal, from the cube project directory

    $ coffee cube.coffee
  4. Open it in your favorite browser

    http://localhost:3000
